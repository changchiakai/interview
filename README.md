# careline-interview-test
CareLine Interview Test

請 clone (或下載) 這個 maven project, 

然後 build 出一個 jar 檔, 

接著執行指令:

java -jar jar檔案路徑

然後打開瀏覽器, 連到 http://localhost:8888, 

依照所看到的頁面提示, 去完成題目, 

題目完成後, 請把程式碼打包成 zip, 

再另外附上履歷檔案, 寄 Email 到 jack.wang@careline.com.tw
